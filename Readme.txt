Для запуска приложения должен быть предустановлен Node.js
Запускаем VS Code и открываем в нём папку проекта.
Открываем терминал и вводим следующую команду для установки всех используемых библиотек в приложении:
npm install
Для запуска приложения вводим эту команду:
npm start
После того как приложения скомпилируется можно его запустить с телефона находясь в одной локальной сети с устройством на котором запущено приложение посредством мобильного приложения "Expo Go" отсканировав QR-код, или же перейдя по ссылке которую выдаст после запуска приложения (так же как и QR-код).
